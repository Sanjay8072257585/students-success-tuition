# Students Success Tuition Centre Management System

## Overview

Students Success Tuition Centre Management System is a web-based application developed using Python Flask. The system helps manage student enquiries, course information, and administrative tasks efficiently.

## Features

- Student Enquiry Form
- Admin Login System
- Admin Dashboard
- Excel Export Functionality
- Responsive Design
- Google Maps Integration
- WhatsApp Contact Button
- SQLite Database Integration

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- Bootstrap
- SQLite
- GitHub
- Render
